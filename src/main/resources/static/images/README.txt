HƯỚNG DẪN THÊM HÌNH ẢNH

1. Tạo một file hình ảnh có tên "hero.jpg" (hoặc tên khác)
2. Đặt file hình ảnh vào thư mục này (src/main/resources/static/images)
3. Nếu bạn sử dụng tên khác cho file hình ảnh, hãy cập nhật đường dẫn trong file CSS:
   - Mở file src/main/resources/static/css/style.css
   - Tìm dòng: background-image: url('/images/hero.jpg');
   - Thay đổi 'hero.jpg' thành tên file hình ảnh của bạn

Lưu ý: Kích thước đề xuất cho hình ảnh hero là 1920x1080px.
